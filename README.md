## 単語学習アプリの設計図

### ユースケース図

```mermaid
flowchart LR
    User([学生・利用者])

    UC1((単語と意味を登録する))
    UC2((登録した単語を一覧表示する))

    UC4((単語情報を編集する))
    UC5((不要な単語を削除する))
    UC6((確認テストを開始する))
    UC7((答えを入力して判定する))
    UC8((覚えた/未学習を切り替える))
    UC9((入力不足時にエラーを確認する))
    UC10((登録データを保存・読み込みする))

    User --> UC1
    User --> UC2
    User --> UC3
    User --> UC4
    User --> UC5
    User --> UC6
    User --> UC7
    User --> UC8

    UC1 -.単語または意味が未入力.-> UC9
    UC1 -.登録後.-> UC10
    UC4 -.更新後.-> UC10
    UC5 -.削除後.-> UC10
    UC7 -.判定結果を反映.-> UC8
    UC8 -.状態変更後.-> UC10
```


### シーケンス図

```mermaid
sequenceDiagram
    actor User as 利用者
    participant UI as 画面
    participant App as 単語学習アプリ
    participant Store as 保存領域

    User->>UI: アプリを開く
    UI->>App: 初期表示を要求する
    App->>Store: 登録済み単語を読み込む
    Store-->>App: 保存済みデータを返す
    App-->>UI: 単語一覧・登録数・学習状態を表示する

    rect rgb(245, 248, 255)
        Note over User,Store: 単語登録
        User->>UI: 単語・意味・品詞・例文を入力する
        User->>UI: 登録するを押す
        UI->>App: 入力内容を送信する
        alt 単語または意味が未入力
            App-->>UI: エラーメッセージを返す
            UI-->>User: 「単語と意味を入力してください。」を表示する
        else 入力内容が正しい
            App->>App: 単語データを作成する
            App->>Store: 単語データを保存する
            Store-->>App: 保存完了
            App-->>UI: 登録結果と最新一覧を返す
            UI-->>User: 登録完了メッセージと一覧を表示する
        end
    end

    rect rgb(248, 255, 245)
        Note over User,Store: 検索・編集・削除
        User->>UI: 検索キーワードまたは状態を指定する
        UI->>App: 検索条件を送信する
        App->>App: 条件に合う単語を抽出する
        App-->>UI: 検索結果を返す
        UI-->>User: 条件に合う単語を表示する

        User->>UI: 編集または削除を押す
        UI->>App: 対象単語の更新または削除を要求する
        App->>Store: 更新後の単語データを保存する
        Store-->>App: 保存完了
        App-->>UI: 最新一覧を返す
        UI-->>User: 更新後の一覧を表示する
    end

    rect rgb(255, 250, 240)
        Note over User,Store: 確認テスト
        User->>UI: 出題形式を選び、問題を出すを押す
        UI->>App: 確認テスト開始を要求する
        alt 登録単語が0件
            App-->>UI: エラーメッセージを返す
            UI-->>User: 「単語を登録してからテストを開始してください。」を表示する
        else 登録単語が1件以上
            App->>App: 出題する単語を選ぶ
            App-->>UI: 問題を返す
            UI-->>User: 単語または意味の問題を表示する

            User->>UI: 答えを入力して判定するを押す
            UI->>App: 入力された答えを送信する
            alt 答えが空欄
                App-->>UI: エラーメッセージを返す
                UI-->>User: 「答えを入力してください。」を表示する
            else 正解
                App->>App: 正解数を増やし、覚えた状態にする
                App->>Store: 更新後の単語データを保存する
                App-->>UI: 正解結果を返す
                UI-->>User: 正解メッセージを表示する
            else 不正解
                App->>App: 不正解数を増やし、未学習状態にする
                App->>Store: 更新後の単語データを保存する
                App-->>UI: 不正解結果を返す
                UI-->>User: 不正解メッセージと正解を表示する
            end
        end
    end
```

### 状態遷移図

```mermaid
stateDiagram-v2
    [*] --> 起動

    起動 --> データ読込中 : アプリを開く
    データ読込中 --> 単語未登録 : 登録データ0件
    データ読込中 --> 単語一覧表示中 : 登録データ1件以上

    単語未登録 --> 登録入力中 : 単語を入力する
    登録入力中 --> 単語未登録 : 単語または意味が未入力
    登録入力中 --> 単語一覧表示中 : 登録成功

    単語一覧表示中 --> 登録入力中 : 新しい単語を入力する
    単語一覧表示中 --> 編集中 : 編集を押す
    編集中 --> 単語一覧表示中 : 更新する
    編集中 --> 単語一覧表示中 : 編集をやめる

    単語一覧表示中 --> 単語一覧表示中 : 検索する
    単語一覧表示中 --> 単語一覧表示中 : 覚えた/未学習を切り替える
    単語一覧表示中 --> 単語一覧表示中 : 単語を削除する
    単語一覧表示中 --> 単語未登録 : 最後の単語を削除する

    単語未登録 --> 単語未登録 : 問題を出す / エラー表示
    単語一覧表示中 --> 出題中 : 問題を出す

    出題中 --> 出題中 : 答えが空欄 / エラー表示
    出題中 --> 正解表示中 : 正解を入力する
    出題中 --> 不正解表示中 : 不正解を入力する

    正解表示中 --> 単語一覧表示中 : 正解数を更新し覚えた状態にする
    不正解表示中 --> 単語一覧表示中 : 不正解数を更新し未学習状態にする

    単語一覧表示中 --> データ保存中 : 登録・編集・削除・状態変更・判定
    データ保存中 --> 単語一覧表示中 : 保存成功
    データ保存中 --> 単語一覧表示中 : 保存失敗 / エラー表示

    単語一覧表示中 --> [*] : アプリを閉じる
    単語未登録 --> [*] : アプリを閉じる
```
